Welcome to Web Page Reader!

Install Text-to-Speech(TTS) Engine

The TTS engine that ships with the Android device supports English, French, German, Italian and Spanish.

If you use these languages you don't have to install the TTS engine.

You can install TTS engines for other languages in Anroid Market. For example


Select the default engine

If you have multiple TTS engines, select the default engine as follows.

  1. Go to Android Settings -> Voice Input & Output -> Text-to-Speech settings.
  2. Activate the installed engine at the bottom of the screen, then select the engine as the Default Engine.

Usage

Start reading
Stop reading
Press the button to fast-forward.
Press the button to rewind.

The current reading position can be changed by the press on the web page.
You can cancel the link navigation by the long press.

There is also Paid version
Web Page Reader Paid Ver.

Compared with this free version, Paid version has the following features.

  • Ads are not displayed.
  • The app keeps reading after another app becomes active or the monitor turns off.
  • The web page can be saved as a text file, so you can read it offline.
  • Image loading can be disabled, so speed up page loading and reduce network cost.
