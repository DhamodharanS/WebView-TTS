TTS(text-to-speech) 엔진 설치

Android 단말기에는 표준으로 영어, 프랑스어, 독일어, 이탈리아어, 스페인어의 TTS(text-to-speech) 엔진이 들어 있습니다.

이러한 언어로 음성시키는 경우 TTS 엔진 설치가 필요하지 않습니다.

일본어로 읽기시키려면, KDDI 연구소의 TTS 엔진 (무료)을 설치합니다.

중국어와 한국어로 읽기시키려면, SVOX 사의 TTS 엔진 (유료)을 설치합니다.


휴대폰 기능 안에 한국어 TTS 엔진이 있을경우에는 설치가 필요하지 않습니다.


TTS 엔진 선택

TTS 엔진이 2 개 이상있는 경우 사용할 엔진을 선택합니다.

  1. Android 단말의 "설정" -> "음성 입력 & 출력" -> "TTS(text-to-speech) 설정" 화면을 엽니다.
  2. 엔진을 설치하면 화면 하단의 엔진의 이름 오른쪽의 확인란을 선택합니다.
  3. 같은 화면에서 "기본 엔진" 을 선택합니다.

사용법

읽기를 시작합니다.
읽기를 중지합니다.
버튼을 길게 눌러 읽기 위치를 앞으로 이동합니다.
버튼을 길게 눌러 읽기 위치를 뒤로 이동합니다.

웹페이지를 터치해 낭독 위치를 지정할 수 있습니다.
터치를 길게 하면 링크로 페이지 이동하는 것을 막을 수 있습니다.

유료 버전도 있습니다.
유료 버전 웹 페이지 리더

무료 버전과 비교하여 유료 버전은 다음과 같은 기능이 있습니다.

  • 광고가 게재되지 않습니다.
  • 다른 응용 프로그램이 활성화되거나 모니터가 꺼진 후에도 계속 읽습니다.
  • 웹 페이지는 텍스트 파일로 저장할 수 있으므로 오프라인으로 읽을 수 있습니다.
  • 이미지 로드를 비활성화할 수 있기 때문에 페이지 로딩을 고속화하며 네트워크 비용을 줄일 수 있습니다.
